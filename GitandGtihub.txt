# PART 6 — Merge Conflict (MOST IMPORTANT TOPIC)

THIS is clearly your faculty’s favorite topic.

Both sample questions are conflict-resolution based.

So understand deeply.

---

# WHY CONFLICT OCCURS

Conflict happens when:

* same file
* same lines
* modified differently
* in different branches

---

# Example

---

## main branch

```text
Login Module: Main branch implementation
```

---

## login-feature branch

```text
Login Module: Initial implementation
```

---

When merging:
Git gets confused.

---

# Conflict Markers

Git adds:

```text
<<<<<<< HEAD
Main branch implementation
=======
Feature implementation
>>>>>>> login-feature
```

VERY IMPORTANT.

---

# HOW TO RESOLVE CONFLICT

You manually edit file.

---

# Expected Final Content

From your sample:

```text
Login Module: Initial implementation
Login Module: Main branch implementation
```

---

# Then:

```bash id="分快三"
git add login.txt

git commit -m "LoginMergeResolved"
```

This finalizes merge.

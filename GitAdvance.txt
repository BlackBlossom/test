This already appeared in your sample paper.

# What is git revert?

It:

* undoes a commit
* WITHOUT deleting history
* creates a NEW reverse commit

SAFE operation.

---

# Syntax

```bash id="0i5d63"
git revert commit_id
```

---

# Example

Suppose:

```text id="njlwmz"

```

Run:

```bash id="2jlwmz"

```


Exactly like your sample.

# PART 3 — Git Reset

VERY IMPORTANT difference from revert.

---

# What is git reset?

Moves HEAD backward.

Can:

* uncommit changes
* remove commits
* remove staging

---

# Types of Reset

| Type    | Effect            |
| ------- | ----------------- |
| --soft  | keep changes      |
| --mixed | unstage changes   |
| --hard  | delete everything |

---

# Soft Reset

```bash id="8jlwmz"
git reset --soft HEAD~1
```

Removes last commit BUT keeps changes staged.

---

# Hard Reset

```bash id="9jlwmz"
git reset --hard HEAD~1
```

DANGEROUS.

Deletes:

* commit
* file changes

---

# VERY IMPORTANT DIFFERENCE

| Revert                 | Reset            |
| ---------------------- | ---------------- |
| safe                   | destructive      |
| keeps history          | rewrites history |
| creates reverse commit | removes commits  |

HIGH probability MCQ.

# PART 4 — Git Rebase (VERY IMPORTANT)

This is a BIG DevOps workflow concept.

---

# What is Rebase?

Moves branch commits on top of another branch.

Creates:

* cleaner history
* linear history

---

# Merge vs Rebase

| Merge                | Rebase          |
| -------------------- | --------------- |
| creates merge commit | cleaner history |
| branching graph      | linear history  |

---

# Example

Suppose:

```text id="ajlwmz"
main:
A---B

feature:
     C---D
```

After rebase:

```text id="bjlwmz"
A---B---C---D
```

---

# Rebase Command

```bash id="cjlwmz"
git rebase main
```

Usually done from feature branch.

---

# IMPORTANT

Rebase can ALSO create conflicts.

Resolve similarly:

* edit file
* git add .
* continue rebase

---

# Continue Rebase

```bash id="djlwmz"
git rebase --continue
```

---

# Abort Rebase

```bash id="ejlwmz"
git rebase --abort
```

